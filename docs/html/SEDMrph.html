Jordi et. al 2006

Transformation

U-B = (0.79 ± 0.02)*(u-g) - (0.93 ± 0.02) U-B = (0.52 ± 0.06)*(u-g) + (0.53 ± 0.09)*(g-r) - (0.82 ± 0.04) B-g = (0.175 ± 0.002)*(u-g) + (0.150 ± 0.003) B-g = (0.313 ± 0.003)*(g-r) + (0.219 ± 0.002) V-g = (-0.565 ± 0.001)*(g-r) - (0.016 ± 0.001) V-I = (0.675 ± 0.002)*(g-i) + (0.364 ± 0.002) if g-i <= 2.1 V-I = (1.11 ± 0.02)*(g-i) - (0.52 ± 0.05) if g-i > 2.1 R-r = (-0.153 ± 0.003)*(r-i) - (0.117 ± 0.003) R-I = (0.930 ± 0.005)*(r-i) + (0.259 ± 0.002) I-i = (-0.386 ± 0.004)*(i-z) - (0.397 ± 0.001)
